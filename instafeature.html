Project

Instagram is a popular social media app that allows users to share interesting photos with the world. As a design challenge, I researched, ideated, wireframed, and prototyped a potential feature that Instagram users would like to have. This feature is simple to use, streamlined with the updated Instagram UI. The new feature is "to create photo collages" in Instagram. It gives users more freedom to capture the moments they love.

My Role

  • Persona
  • User Stories
  • User Flows
  • Wireframes
  • Interface Design
  • Prototype

Epics

  • Users can create image collages
  • Users can edit image collages
  • Users can view image collages
  • Users can share image collages
  • Users can delete image collages

Design Stories

  • Users can:
  • take multiple pictures and use them to create a photo collage
  • select multiple pictures and use them to create a photo collage
  • take pictures and select pictures to create a photo collage
  • add filters to each picture of the photo collage
  • crop and scale each pictures that are selected
  • choose collage templates
  • choose how to position each picture in the photo collage
  • replace each picture in the collage with another photo/li>
  • delete any pictures added into the collage
  • view collages
  • preview collage before sharing
  • cancel making a photo collage
  • add captions to the collage
  • add location, tag people in the collage
  • share the collage

User Flow

Considering that this "photo collage" feature could start from various places in the app, I had thought about where to place it for a long time. Based on my understanding and my personal experience, I define Instagram as more of a camera, rather than a photo editing app. That being said, I chose to add my feature inside Instagram's "Tool" section.
